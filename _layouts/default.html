Projects

SecurityReview.ai

Agentic AI, threat modeling, RAG, OSCAL

SecurityReview.ai turns architecture documents, diagrams, tickets, code, and other engineering artifacts into system-specific threat models, countermeasures, and compliance-ready review outputs. I architected core workflows for multi-modal ingestion, code profiling, specialized review agents, PWNISMS category analysis, threat-intelligence retrieval, and OSCAL-backed compliance mapping.

VibeReview

AI coding guardrails, MCP, GitHub, GitLab, IDE workflows

VibeReview is a SaaS guardrail platform for AI-assisted development. It profiles repositories, generates threat-informed rules, and delivers them into developer workflows through MCP, GitHub, GitLab, and IDE integrations so AI coding tools get security context before risky changes are made.

KG-MuLQA

Knowledge graphs, long-context LLM evaluation, NLP

KG-MuLQA builds document-level knowledge graphs from SEC credit agreements and generates multi-level QA benchmarks that test relation hops, set operations, and answer plurality. The benchmark covers 20,139 questions over 170 credit agreements and is used to study long-context LLM failure modes.
